# Getting Started

Welcome to the Markdown documentation site.

## Introduction

This project automatically converts Markdown files into HTML pages using Python.

## Features

- Markdown to HTML conversion
- GitHub Pages deployment
- Automatic CI/CD with GitHub Actions
- Clean CSS styling

## Installation

Run the following command:

```bash
pip install markdown
```

## Project Structure

